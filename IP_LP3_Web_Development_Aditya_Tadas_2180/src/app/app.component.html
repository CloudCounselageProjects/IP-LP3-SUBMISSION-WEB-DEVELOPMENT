LP3 Assignment - Web Development
Problem:
Create a personal website using full-stack development which uses a REST API to tell system date and time by an API call. The website should work on the localhost.
Solution Details:
  • Frontend: Angular
  • Backend: Express-Node.js
  • Localhost port: 8888
  • Site running on: http://localhost:8888/
  • REST API call: http://localhost:8888/api/datetime
